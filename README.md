# Public Transportation Analytics Dashboard

## Overview
The Public Transportation Analytics Dashboard is a web-based application designed to help urban planners, policymakers, businesses, and residents make data-driven decisions for optimizing the public transportation system in Abu Dhabi. The dashboard leverages the "Public Transportation Ridership Data - Abu Dhabi" dataset to provide interactive visualizations, predictive analytics, and sustainability metrics.

## Features
- **Interactive Heatmaps**: Visualize the most utilized routes and peak hours.
- **Monthly Ridership Trends**: Understand ridership dynamics over time.
- **Demographic Insights**: Analyze passenger demographics.
- **Predictive Analytics**: Forecast future ridership trends.
- **Data Export**: Download data in CSV, Excel, or JSON formats.

## Installation
1. Clone the repository:
   bash
   git clone https://github.com/username/public-transportation-dashboard.git
   
2. Navigate to the project directory:
   bash
   cd public-transportation-dashboard
   
3. Install dependencies:
   bash
   pip install -r requirements.txt
   

## Usage
1. Place the "Public_Transportation_Ridership_Data.csv" file in the project directory.
2. Run the script to analyze the data:
   bash
   python analyze_data.py
   
3. View the generated visualizations and predictions in the output directory.
